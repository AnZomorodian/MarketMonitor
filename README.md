# Just Check Market

A real-time market price tracker displaying cryptocurrency, gold/coins, and currency exchange rates.

## Features

- **Live Crypto Prices** - Track Bitcoin, Ethereum, and 20+ cryptocurrencies
- **Gold & Coins** - Monitor gold prices, Azadi coins, and precious metals with visual chart
- **Currency Exchange** - View exchange rates for major world currencies
- **Auto-Refresh** - Prices update automatically every 5 minutes
- **Modern UI** - Clean, responsive design with dark theme

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Framer Motion
- **Backend**: Express.js, TypeScript
- **Charts**: Recharts
- **Build**: Vite

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open the app at http://localhost:5000
